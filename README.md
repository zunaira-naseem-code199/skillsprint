# SkillSprint 🚀

Personal Learning & Career Growth Platform

## Description

SkillSprint is a MERN stack application that helps students and self-learners organize their learning journey, track goals, manage tasks, maintain learning journals, and showcase certificates.

## Tech Stack

Frontend:
- React
- Vite
- React Router
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Future Features:
- User Authentication
- Goal Management
- Task Management
- Learning Journal
- Certificates
- Dashboard Analytics
- AI Suggestions (Future Enhancement it's optional)
